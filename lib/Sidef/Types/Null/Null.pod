=head1 NAME

Sidef::Types::Null::Null - The Null type in Sidef

=head1 DESCRIPTION

This class implements the B<Null> type in Sidef, representing the absence of a value or an undefined state. Null is a singleton object that serves as Sidef's representation of "nothing" or "no value". It is commonly used as a default return value for operations that don't produce meaningful results, or to indicate the absence of data.

The Null object evaluates to false in boolean contexts and can be used for nil-checking patterns common in many programming scenarios.

=head1 SYNOPSIS

    # Null is automatically returned by some operations
    var result = [1,2,3].first { _ > 10 }  # returns Null if not found
    
    # Explicit Null creation
    var obj = Null()
    var nothing = Null.new
    
    # Boolean context - Null is falsy
    if (Null) {
        say "This won't execute"
    }
    
    # Checking for Null
    if (result == Null) {
        say "No result found"
    }
    
    # Using Null in conditionals
    var value = some_function() \\ "default"  # Use default if Null
    
    # Converting Null to string
    say Null.to_s  # prints "null"

=head1 INHERITS

Inherits methods from:

       * Sidef::Object::Object

=head1 METHODS

=head2 new

    Null.new
    self.new

Returns the singleton Null object. Since Null is a singleton, all calls to C<new> return the same Null instance.

B<Example:>

    var n1 = Null.new
    var n2 = Null.new
    say (n1 == n2)  # true - same object

Aliases: I<call>

=cut

=head2 to_s

    Null.to_s
    self.to_s

Returns the string representation of the Null object, which is the lowercase string C<"null">.

B<Example:>

    say Null.to_s        # prints: null
    say "Value: #{Null}" # prints: Value: null

B<Returns:> A String object containing C<"null">

Aliases: I<dump>, I<to_str>

=cut

=head1 BOOLEAN CONTEXT

The Null object evaluates to false in boolean contexts:

    if (Null) {
        # This block never executes
    }
    
    say (Null ? "true" : "false")  # prints: false
    say (!Null)                     # prints: true

=head1 EQUALITY AND COMPARISON

Null can be compared with other values:

    say (Null == Null)      # true
    say (Null == nil)       # true (nil is an alias)
    say (Null == false)     # false
    say (Null == 0)         # false
    say (Null == "")        # false

=head1 SYNOPSIS

=head1 DESCRIPTION

The C<Sidef::Types::Glob::File> class represents a filesystem path and provides
methods for inspecting, reading, writing, and manipulating files and
directories.

A C<File> object behaves like a string containing a path, while also exposing
filesystem-related functionality. This allows file paths to be used naturally
as strings while enabling direct interaction with the filesystem.

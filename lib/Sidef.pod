=head1 NAME

Sidef - The Sidef programming language.

=head1 DESCRIPTION

Sidef is a powerful programming language inspired by B<Ruby>, B<Raku>, and B<Go>. It is designed to be expressive and flexible, seamlessly blending functional and object-oriented paradigms.

=head2 Key Features

=over 4

=item * B<Object-Oriented Programming:> Full support for classes, objects, and inheritance.

=item * B<Functional Programming:> Includes functional pattern matching, first-class functions, and lexical closures.

=item * B<Advanced Dispatch:> Supports multiple dispatch and keyword arguments.

=item * B<Perl Integration:> Full support for utilizing existing Perl modules directly within Sidef.

=item * B<Rich Math Library:> Native support for arbitrary-precision integers, rationals, floats, and complex numbers.

=item * B<Control Flow:> Features optional lazy evaluation and integrated regular expressions.

=item * B<Safety & Scoping:> Includes lexical scoping and optional dynamic type checking.

=back

=head1 SYNOPSIS

The C<Sidef> module allows you to interface with the language engine directly from Perl code.

    use Sidef;

    # Initialize the Sidef engine
    my $sidef = Sidef->new(
        name       => 'test',   # name of the program
        opt        => {},       # command-line options
        parser_opt => {},       # parser options
    );

    # Define Sidef source code
    my $code = 'say "1 + 2 = #{1 + 2}"';

    # 1. Execute directly
    $sidef->execute_code($code);

    # 2. Transpile to Perl
    my $perl = $sidef->compile_code($code, 'Perl');

    # 3. Advanced AST manipulation
    my $ast  = $sidef->parse_code($code);          # Generate AST
    my $oast = $sidef->optimize_ast($ast);         # Optimize (constant-folding)
    print $sidef->compile_ast($oast, 'Sidef');     # Compile back to Sidef source

=head1 METHODS

=head2 new

    my $s = Sidef->new(name => '...');

Creates and returns a new C<Sidef> object.

=head2 execute_code

    my $result = $s->execute_code($code);

Parses, compiles, and executes Sidef code. Returns the result of the final statement executed.

=head2 compile_code

    my $compiled_code = $s->compile_code($code, $backend);

Parses and compiles the source string into the target backend format.
B<Valid backends:> C<'Sidef'>, C<'Perl'>.

=head2 parse_code

    my $ast = $s->parse_code($code);

Parses the source code and returns the Abstract Syntax Tree (AST).

=head2 optimize_ast

    my $oast = $s->optimize_ast($ast);

Performs constant-folding and other optimizations on an AST, returning a new optimized tree.

=head2 compile_ast

    my $compiled_code = $s->compile_ast($ast, $backend);

Compiles an existing AST into code for the specified backend.
B<Valid backends:> C<'Sidef'>, C<'Perl'>.

=head2 version

    my $version_str = Sidef->version;

Returns the Sidef version as a B<String> object.

=head2 numeric_version

    my $version_num = Sidef->numeric_version;

Returns the Sidef version as a B<Number> object.
